Detection of Blighted Houses

Develop a classification model detecting blighted houses.

By Kwang Ho Hur at The Data Incubator


Issue and Motivation

Blighted houses or buildings are problematic in a lot of cities because of

  • Threat to public safety
  • More blighted houses can be created in the neighborhood.
  • Concern for real estate value

The detection of blighted houses is important because of

  • Useful information for insurance companies
  • New opportunity for real estate developers

Blighted House in Detroit

    (Figure 1) An example of Blighted House

  • This issue has been much more serious in Detroit because of the enormous decline of its population during the last 60 years. In the city, lots of houses have been abandoned. This enables me to build a classification model where the number of blighted houses is high enough to be detectable.

    • Population of Detroit and USA

    Solution

    1. Build a classification model which can identify blighted houses in the city.
    2. Evaluate the accuracy of the model
    3. Develop a method employing images to improve the model

    Data

    • Phase 1 Using public data from Detroit
      • Demolition Permits
      • Blight Violation Tickets
      • Crimes
      • 311 Calls

    Distribution of Blighted Houses in Detroit

      (Figure 3) Blighted houses are distributed in a whole region of Detroit (drawn in CARTO)

    Classification Model

      - K-Nearest Neighboring Classification model was built with features of Latitude, Longitude , the number of blight violation tickets, the number of crimes, and the number of 311 calls. The parameter k of the best model is 6.
